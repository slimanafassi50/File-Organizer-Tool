# File Organizer Tool

A simple and efficient desktop application built with Python to help you organize your files automatically. This tool sorts your files into specific categories (Images, Documents, Scripts) based on their extensions, helping you keep your workspace clean.

## Features
- **Automatic Sorting:** Automatically detects file extensions and moves them into organized folders.
- **User-Friendly Interface:** Built using `tkinter` for a smooth and intuitive graphical experience.
- **Error Handling:** Safely handles file operations with built-in alerts.
- **Customizable:** Easily extendable to include more file types and categories.

## How it Works
1. When you run the application, a simple window will appear.
2. Click the **"Select Folder and Start"** button.
3. Choose the directory you want to organize.
4. The tool will automatically create subfolders and move your files into their respective categories.

## Built With
- **Python** - The core programming language.
- **Tkinter** - Used for the graphical user interface.
- **os & shutil** - Used for file system manipulation and moving files.
